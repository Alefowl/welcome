Добавление книги:
1. На странице My account нажать кнопку Add book.
2. На странице Add book заполнить поля формы.
3. Нажать кнопку Add book внизу формы.

Добавление текста:
1. На странице My account нажать кнопку Add text.
2. На странице Text of a book в поле The book of this text можно выбрать
добавленную книгу.
3. При нажатии кнопки Check в поле для ввода текста появляется сообщение "The
book exists, but".
4. Добавить текст книги в соответствующее поле.
5. Нажать кнопку Mark up.
6. Проверить появившуюся разметку, поправить при необходимости.
7. Нажать кнопку Add text.

После выполнения всех действий информация о добавленной книги появляется в
разделе My books. Книга классифицируется как "неопубликованная с текстом".

Рекомендации по улучшению процесса разметки и добавления книги:
1. Для удобства выбора книг в разделе Stats добавить возможность поиска, либо
расположить строки в алфавитном порядке.
2. Для удобства процесса добавления текста в течение нескольких дней в разделе
Add text добавить возможность сохранения уже набранного текста.
3. Добавить пояснение к функциям конопок Check и Mark up.
