6. Downloads
PASSED:
Переход со страницы My account для просмотра отправленных пользователю книг осуществляется нажатием кнопки Downloads. Список отсортирован по очередности получения книг, начиная с более поздних. При переходе по ссылке на название книги осуществляется переход на страницу загрузки книги.
Повторный запрос на monolingual book осуществляется со страницы Content кнопкой Send this book via email to ...
Повторный запрос на bilingual books осуществляется со страницы загрузки книги кнопкой Send this book via email to ...
Повторный запрос на snippet of bilingual books осуществляется со страницы загрузки книги кнопкой Send the snippet (only the current fragment) of this book via email to ...

7. Добавление книги

PASSED:
Переход со страницы My account на страницу добавления книги осуществляется нажатием кнопки Add book.
Поля The title и The author принимают символы, в том числе ! @#$%-^&*()_+~"№;%:?=[]{}|/,.'<>, кириллические, латинские, древнегреческие, китайские символы, символы языка хинди и арабской вязи, арабские и римские цифры. Максимально допустимое количество символов не отображается.
Поле Year of translation принимает арабские цифры.
В поле The metabook for this book открывается выпадающий список (во время проверки список метакниг для загрузки был недоступен).
Выбор языка для загрузки осуществляется из выпадающего списка в поле The language of a book. Выбранный язык отображается корректно.
Переход со страницы My account на страницу Text of a book осуществляется нажатием кнопки Add text.

BUG:
Нижняя граница кнопки Add text скрыта под блоком footer. VISUAL
