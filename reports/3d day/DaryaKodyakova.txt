Тестирование функциональности.

Задача: проверить выполнение входа на сайт в статусе Contributor.
Выполнение:
При нажатии на кнопку Log In в верхней правой части главной страницы осуществляется переход на страницу
http://www.alefowl.com/auth/authentication с формой. Форма содержит поля для заполнения Enter your login,
Enter your pass и кнопки Log in, Sign Up, Forgot password.

При наведении курсора на поле для заполнения Enter your login из всплывающего окна можно выбрать уже созданный аккаунт.
При клике ЛКП поля Enter your login и Enter your pass заполняются автоматически.

При нажатии на кнопку Log in осуществляется переход на страницу с формой My account (verified).

Тест пройден.

Задача: проверить функционирование кнопок в разделе My account в статусе Contributor.
Выполнение:
При нажатии кнопки My settings открывается форма с полями данных, заполненных ранее при регистрации.

При нажатии кнопки My books осуществляется переход на страницу, содержащую сведения о загруженных книгах
(при данном тестировании книги не загружались). Страница содержит следующие наименования: Id, title (author) (language);
Owner; Publish book; Delete book; Edit content. Ниже на странице приведены пояснения к цветовым обозначениям книг.

При нажатии кнопки Stats осуществляется переход на страницу, содержащую сведения о книгах в виде таблицы с заголовками:
Author; Metabook; Lift; Delete; доступные языки. В столбце Author содержатся кликабельные ссылки, при нажатии
на которые, осуществляется переход на страницу соответствующего автора. В колонках, соответсвующих определенному языку,
приведены кликабельные ссылки с числами и цветовым обозначением. Внизу страницы приведены пояснения к цветовым
обозначениям книг.

При нажатии кнопки Downloads осуществляется переход на страницу The downloaded books, содержащую сведения о загруженных
книгах в виде таблицы с заголовками: The first book; The second book; Date.

При нажатии кнопки Add book осуществляется переход на страницу с формой для заполнения данных. При нажатии
кнопки Add book внизу формы и наличием не заполненных полей появляется всплывающая подсказка "Заполните это поле"
рядом с не заполненным полем.

При нажатии кнопки Add text осуществляется переход на страницу Text of a book. Под заголовком странциы есть поле
The book of this text с выпадающим меню. При данных условиях тестирования (нет добавленных книг,
нет добавленного текста) выпадающее меню пустое. Ниже находится поле для заполнения текстом. Под полем находятся кнопки
Mark up и Add text. При нажатии кнопки Mark up или Add text и отсутствии текста появляется всплвывающее окно браузера
с ошибкой: "Подтвердите действие на странице www.alefowl.com Text field is empty. Please enter some text.

При нажатии кнопки My account в правом верхнем углу страницы осущетсвляется переход на страницу аккаунта.

При нажатии кнопки Log out в правом верхнем углу или внизу страницы осуществляется выход из аккаунта.

Результат: все кнопки функционируют корректно. загрузка форм осуществляется корректно.

Рекомендация: на странице Stats добавить информацию, раскрывающую значение чисел, приведенных в колонках,
соответствующих определенному языку.

Тест пройден.
