Общее поведение:
Сайт загружается корректно, перемещение по элементам с клавиатуры и скроллинг работают нормально. Ссылки отрабатывают корректно. Баги и рекомендации по устранению описаны ниже.



Дефекты и рекомендации по их устранению.

Главная страница - Шапка сайта
Проблема с кликом по логотипу. 
не реализована единая ссылка на весь логотип. Вместо этого на текстовую часть логотипа ("Alefowl" "Bilingual Books") навешены 2 отдельные ссылки, выполняющие одно и тоже действие. Изображение в логотипе некликабельное. Висит отдельным элементом.
Рекомендация: реализовать одну единую ссылку на весь логотип. 

Отсутствует возможность создания учетной записи непосредственно с главной страницы. 
На главной странице отсутствует ссылка для регистрации (есть только ссылка для авторизации "Log in"). Ссылка для регистрации запрятана в окне авторизации.
Текст ссылки регистрации в одном окне "Sign in", в другом окне "Register". Когда одно и тоже называется разными терминами, у пользователей возникает путаница.
Рекомендация: вынести ссылку для регистрации на главную страницу.  Оставить только один вариант текста в ссылки для регистрации.

Все перечни книг:
Эффект при наведении реализован только для названия книги. При наведении на фамилию и имя автора в скобках никакого эффекта кроме cursor:pointer нет.
Рекомендация: реализовать эффект наведения на имя автора более явно для пользователя, например аналогично эффекту наведения на название книги.

Подвал:
На пунктах меню отсутствует эффекта при наведении кроме cursor:pointer.
Рекомендация: реализовать эффект наведения более явно для пользователя, например изменением цвета текста.

Внутри пунктов меню "About Us", "Our Team", "Internship" на всех ссылках отсутствует эффект при наведении кроме cursor:pointer.
Рекомендация: реализовать эффект наведения на ссылки более явно для пользователя.

Окно ввода пароля и регистрации:
При наведении курсора на "Sign up" отодвигается соседняя кнопка "Log in".
Кнопка "Log in" реализована тегом input.
Рекомендация: исправить ошибку стилизации. Заменить input на button для отправки формы.
 	


После перехода к конкретной книге, например: 
http://www.alefowl.com/1944/1944/0/

-Селекторы выбора книги: 
-при наведении эффект напоминает кнопки, расположенные ниже.  
-при раскрытии списка стрелка справа не изменяет направление (по-прежнему указывает вниз, а должна указывать вверх).
Рекомендация: убрать эффект при наведении на селектор, обеспечить поворот стрелки на 180 градусов.
-

Два элемента с текстом: 
"Please, log in to get this snippet (only the current fragment) in pdf"
и
"Please, log in to download this book in pdf"
реализованы в виде кнопок, внутрь которых помещены ссылки, навешенные на "log in", что недопустимо. 
В результате отрабатывает лишь кнопка, по щелчку на которую открывается модальное окно, а ссылка на страницу авторизации не отрабатывает вовсе.
Рекомендация: пересмотреть механизм работы элементов, реализовать валидную верстку.

Модальное окно "Downloading books is available only for authorized users":
Кнопка закрытия модального окна "X" растянута и кликабельна по всей ширине модального окна. Должна быть кликабельна только в области значка закрытия.
Модальное окно не закрывается с клавиатуры (по ESC). Зато закрывается с помощью клавиши пробела, поскольку копка остается в фокусе после открытия модельного окна.
Уменьшить ширину кликабельной зоны кнопки до размеров зоны вокруг значка.
Рекомендация: проверить механизм закрытия модального окна с клавиатуры на соответствие требованиям ТЗ.

Кнопки "Sign up" и "Log in" имеют разную ширину, определяемую контентом. Должны быть одинаковой ширины.
Рекомендации: сделать кнопки одинаковой ширины вне зависимости от контента.    
Навигация по книге:
Навигация по двуязычной книге предусмотрена только к следующей или предыдущей главе. Нет возможности вернуться к конкретной главе или перейти к началу/концу книги.
Рекомендация: добавить элементы управления навигацией по книге.

Низкая контрастность подвала.
Рекомендация: добавить атрибут lang в тег html,
поднять контрастность текста и фона подвала.
