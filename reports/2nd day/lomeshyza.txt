Регистрация:

Отчет об ошибках:
Заголовок: Неверный статус регистрации.
Шаги воспроизведения: 
- Открыть страницу по ссылке https://www.alefowl.com.
- Нажать кнопку "Log In" вверху страницы.
- Нажать кнопку "Sign up".
- Заполнить форму.
- Нажать кнопку "Register".

ОР: Происходит регистрация, показывается сообщение об успешной регистрации. 
ФР: Происходит регистрация, показывается сообщение "При регистрации произошла ошибка. Возможно пользователь с таким логином уже сужествует.", но после обновления страницы переадресация в личный кабинет зарегистрированного пользователя.
Дополнение: В поля ввода можно ввести только латинские буквы, при вводе кириллицы , символов или цифр показывается окно ошибки.

Личный кабинет:

Отчет о тестировании:
Заголовок: Функционал личного кабинета.
ОР: По клику на кнопку "Repeat verification" отправляется повторное письмо на почту, для подтверждения email.
    По клику на "My settings" открываются данные о пользователе с возможностью изменения. 
    При нажатии на "Log out" происходит выход из личного кабинета,переадресация на главную страницу незарегистрированого пользователя. 
    При нажатии на кнопку "Downloads" переход на страницу с загруженными книгами.
ФР: Соответствует ожидаемому, за исключением: По клику на "My settings" открываются данные о пользователе с возможностью изменения, кроме графы "Faculty".
Дополнение: В поля ввода можно ввести только латинские буквы, при вводе кириллицы , символов или цифр, пустой строки - показывается окно ошибки.
