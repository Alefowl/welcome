Дата тестирования: 10.07.2024
ОС: Ubuntu 23.04 (lunar)
Браузер: Google Chrome 124.0.6367.60
Условия: зарегистрированный пользователь

Тестирование в ручном режиме

1.	Открытие аккаунта
	Шаги: Клик по кнопке "My account"
	ОР: Открытие страницы с информацией об аккаунте
	ФР: Тест пройден
2. 	Скачивание книги
	Шаги: Открыть главную страницу
	      Зайти в раздел "english books"
	      Кликнуть по любой книге
	      Кликнуть "Download this book in pdf"
	ОР: Появление прогресс-бара при загрузке книги
	    Уведомление об успешной загрузке. Пользователь остается на той же странице.
	ФР: Автоматическое открытие pdf вместо текущего окна сайта.
3. 	Трекер скачанных книг в профиле
	Шаги: Открыть главную страницу
	      Зайти в раздел "english books"
	      Кликнуть по любой книге
	      Кликнуть "Download this book in pdf"
	      Кликнуть "My account"
	      Кликнуть "Downloads"
	ОР: Только что загруженная книга должна отобразиться в списке.
	ФР: Тест пройден.
4. 	Проверка "Log out"
	Шаги: Открыть главную страницу
	      Кликнуть "My account"
	      Кликнуть "Log out"
	ОР: Пользователь попадает на главную страницу, не авторизованный
	ФР: Тест пройден.
5. 	Проверка изменения пользовательских данных.
	Шаги: Открыть главную страницу
	      Кликнуть "My account"
	      Кликнуть "My settings"
	      Изменить произвольное поле
	      Нажать "Update data"
	ОР: Успешное изменение в БД, отображение новых данных в My settings
	ФР: Тест пройден.
6. 	Повторение верификации
	Шаги: Открыть главную страницу
	      Кликнуть "My account"
	      Кликнуть "Repeat verification"
	ОР: На почту высылается повторное верификационное письмо, пользователь больше не может залогинится
	ФР: Тест пройден.
7.  	Изменение факультета
	Шаги: Открыть главную страницу
	      Кликнуть "My account"
	      Кликнуть "My settings"
	      Открыть код страницы
	      Изменить свойство select на с disabled на enabled.
	      Выбрать произвольный факультет.
	      Нажать "Update data"
	ОР: Должен выдавать ошибку или ничего не должно происходить, так как пользователь
		не может сам менять свой факультет.
	ФР: Произошло изменение факультета.
	Замечание: На стороне бекенда проверять права пользователя изменять свой факультет.
