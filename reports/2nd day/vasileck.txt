Дата проведения тестирования: 21.06.2024
Сайт: [https://www.alefowl.com](https://www.alefowl.com/)
Статус пользователя: Зарегистрированный пользователь
Система: Windows 10 Pro
Браузер: Google Chrome Версия 126.0.6478.115 (Официальная сборка), (32 бит)
Тестирование в ручном режиме

1. Переход на страницу регистрации. 
Переход осуществляется верно. Открывается страница с полями для заполнения. 

2. Ввод данных.
Ко всем полям есть доступ. При вводе занятого логина вылезает сообщение, о том, что такой пользователь уже существует. При вводе неверной почты вылезает сообщение о недопустимости такого ввода. При верном заполнении всех трех полей и нажатии кнопки Register открывается страница с данными аккаунта. 

3. Заполнение данных аккаунта.
При нажатии кнопки с настройками открывается форма с полями ввода личных данных. Все поля заполняются в ручную. Вводить можно как числа, так и спецсимволы, данные не разделяются на только буквенные или только числовые. При нажатии кнопки Update Data введенные данные сохраняются. При этом, если ввести логин существующего аккаунта никаких конфликтов не будет. 

4. Верификация аккаунта.  
На странице с данными аккаунта, также присутствует кнопка верификации аккаунта. При нажатии отправляется письмо на почту. При переходе по ссылке с подтверждением открывается ссылка со входом на сайт. Только после этого появляется надпись, что аккаунт подтвержден. 

5. Страница загрузок.
В данных аккаунта есть кнопка Downloads, которая переводит на страницу со скачанными книгами. Если никаких загруженных книг нет, то выводится сообщение "The downloaded books will be displayed here.", если уже происходила загрузка какой-либо книги, то на странице списком будут выведены эти книги с датой загрузки и кнопкой Download. При нажатии выводится сообщение, что книга будет отправлена на почту.

6. Главная страница.
После регистрации аккаунта сверху появляется кнопка My account, которая ведет на страницу с данными аккаунта, и кнопка Log Out, которая производит действие выхода  из аккаунта. Главная страница выглядит также, как и для незарегистрированного пользователя, за исключением, отсутствует кнопка войти в аккаунт.

7. Страница с книгой.
Перейдя на страницу с книгой, становятся доступны новые функции. Появляется возможность скачать фрагмент книги и скачать целиком. В случае скачивания фрагмента книги, открывается поле, где можно выбрать словарь для книги. Ни один из них недоступен для скачивания, фрагмент можно открыть только без словаря. Далее в обоих случаях открывается ссылка на blob, откуда идет дальнейшее скачивание. 

8. Проверка ссылки восстановления пароля.
При нажатии кнопки Forgot Password, появляется поле с вводом почты. Если ввести несуществующую, появится сообщение об ошибке. При вводе верной на почту отправляется временный пароль и ссылка на смену пароля. При заполнении всех полей вылезает сообщение с успешным изменением пароля.
